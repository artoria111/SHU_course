goods.bin------------------------------------------------
八个字节 long long 表示商品数量
八个字节 long long 表示特殊的字节数（默认为0），为预留的

对于每一个商品
八个字节 long long 表示商品编号
四个字节 int 商品名称所占字节
n个字节 string 商品名称
四个字节 int 商品描述所占字节
n个字节 string 商品描述
八个字节 double 商品价格
八个字节 long long 商品件数
八个字节 long long 商家编号

buyer.bin------------------------------------------------
八个字节 long long 表示买家数量
八个字节 long long 表示特殊的字节数（默认为0），为预留的

对于每一个买家
四个字节 int 买家编号所占字节
n个字节 string 买家编号
四个字节 int 买家姓名所占字节
n个字节 string 买家姓名
四个字节 int 买家买的商品件数
对于每一件商品 八个字节 long long 商品编号

seller.bin------------------------------------------------
八个字节 long long 表示卖家数量
八个字节 long long 表示特殊的字节数（默认为0），为预留的

对于每一个卖家
八个字节 long long 表示卖家编号
四个字节 int 卖家姓名所占字节
n个字节 string 卖家姓名
四个字节 int 卖家联系方式所占字节
n个字节 string 卖家联系方式

四个字节 int 卖家卖的商品件数
对于每一件商品 八个字节 long long 商品编号
